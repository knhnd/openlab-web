[React (React.js)](https://ja.react.dev/) とは、Meta (旧 Facebook) が開発した `JavaScript` の UI 構築ライブラリである。

### Overview

#### React

Web のフロントエンドを `HTML`, `CSS`, `JavaScript` のみで開発すると、DOM 操作を行う必要があり、コードが長くなったときに難解になる上にレンダリング (描画) の効率が悪くなる。
そこで、`React` では、「`コンポーネント`」と「`仮想DOM`」という２つによって直感的な開発を行えるようにしている。

- **コンポーネント** : ヘッダーやフッター、メインのコンテンツといった Web ページを構成するパーツをコンポーネントという単位で扱い、それらを組み合わせることでページを構築する
- **仮想 DOM (Virtual DOM)** : 画面を効率的に描画・更新するための仕組みで、Web ページの DOM を JavaScript から直感的に扱えるように仮想的に生成した DOM

Web のフロントエンド開発を行う際に、`React` という UI 構築ライブラリを導入することで、HTML, CSS, JavaScript だけで書くよりも効率的に開発できるようになる。詳しくは [React 公式ドキュメント](https://ja.react.dev/learn)を参照する。

#### Next.js

現在、React を導入する際には、React の細かい設定や複雑な仕組みを省略するために「フレームワーク」を使うことが公式で推奨されている。
React のフレームワークにも様々なものがあるが、このドキュメントでは代表的な React のフレームワークである **[Next.js](https://nextjs.org/)** を利用する。
