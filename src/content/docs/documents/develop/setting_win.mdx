Windows を対象とした開発環境の構築

#### システム環境設定

##### 設定

- アクセシビリティ > マウスポインターとタッチ
    - サイズ: 3
- アクセシビリティ > マウスポインターとタッチ > タッチパッド
    - カーソル速度: MAX

##### コントロールパネル

- 表示方法 > 大きいアイコン > キーボード
    - 表示までの待ち時間: 短く (MAX)
    - 表示の間隔: 早く (MAX)

##### ファイルエクスプローラ

- 表示 > 表示
    - ファイル拡張子を表示する
    - 隠しファイルを表示する

##### キーボード関連

Microsoft Store から [Microsoft PowerToys](https://learn.microsoft.com/ja-jp/windows/powertoys/) をインストール

#### ソフトウェア

##### [Google Chrome](https://www.google.com/intl/ja_jp/chrome/)

- ユーザ数が多いので開発するなら基本は Chrome

##### [Google 日本語入力](https://www.google.co.jp/ime/)

- 環境設定で句読点を「**, .**」に変更 (情報系の論文を書く人向け)

#### 開発系

##### [Windows Subsystem for Linux (WSL)](https://learn.microsoft.com/ja-jp/windows/wsl/)

- Windows 上で Ubuntu などの Linux を実行するためのソフトウェア
