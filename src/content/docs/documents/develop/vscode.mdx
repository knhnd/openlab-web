Microsoft が開発した軽量で高機能なテキストエディタ「[Visual Studio Code (VSCode)](https://code.visualstudio.com/)」のおすすめ拡張機能

#### VSCode の便利機能

- コマンドパレット (VSCode をコマンド操作)

  - Windows : `[Ctrl]+[Shift]+[P]`
  - macOS : `[Cmd]+[Shift]+[P]`

- 設定 (VSCode 自体の設定)
  - Windows: `[Ctrl]+[,]`
  - macOS : `[Cmd]+[,]`
